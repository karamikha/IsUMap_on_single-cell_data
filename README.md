Этот репозиторий содержит код и данные для курсовой работы по теме "Критический анализ и уточнение UMAP". Работа посвящена методам снижения размерности, в частности новому способу IsUMap.

Важные файлы:

- IsUMap.py - файл, содержащий в себе класс с реализацией метода снижения размерности IsUMap.
- metrics.py - файл с функциями для подсчёта метрик: корреляция Спирмана попарных расстояний, kNN-accuracy, kNN-recall.
- *.ipynb - ноутбуки с экспериментами методов t-SNE, UMAP и IsUMap на разных датасетах.
